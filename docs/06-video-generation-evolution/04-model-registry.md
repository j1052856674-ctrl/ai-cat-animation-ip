## 快速切换指南

```bash
# 切换到性价比优先（默认）
export MODEL_PRESET=default

# 切换到质量优先
export MODEL_PRESET=quality-first

# 切换到成本优先（测试阶段）
export MODEL_PRESET=cost-first

# 切换到 A/B 测试
export MODEL_PRESET=ab-test

# 临时覆盖某个模型
export SCRIPT_MODEL_OVERRIDE=claude-3-5-sonnet
export IMAGE_MODEL_OVERRIDE=midjourney-v6
export VIDEO_MODEL_OVERRIDE=kling-api
```
