# 目标粉丝画像

## 核心粉丝假设

| 维度 | 画像 |
|------|------|
| 年龄 | 22-35岁 |
| 城市 | 一二线城市（北上广深杭成渝武南等） |
| 职业 | 互联网/金融/广告/设计/新媒体等脑力劳动行业为主，泛打工人 |
| 收入 | 月入6k-25k，有"努力但没富"的共鸣 |
| 生活状态 | 租房或刚买房，通勤30-60分钟，外卖频率高 |
| 社交习惯 | 小红书/抖音重度用户，朋友圈偶尔发、微博潜水 |
| 消费特点 | 愿意为情绪价值小额付费（奶茶、盲盒、游戏皮肤、IP周边），大件消费理性 |

## 核心需求

| 需求层级 | 具体表现 | 阿饱如何满足 |
|----------|----------|-------------|
| 情绪宣泄 | 职场压力需要出口，但朋友圈不能说、微博不想说 | 阿饱替他们说——吐槽无风险、转发无负担 |
| 被理解感 | "我不是一个人"——加班、改需求、周一不想起床，需要同类确认 | 场景精准共鸣 → "这说的就是我" |
| 微小治愈 | 在疲惫中需要一点温暖，不需要大道理 | 阿饱吃饭 = 提醒用户"对自己好一点" |
| 身份认同 | "我是打工人但我不可怜"——需要一种不愤怒、不自怜的表达 | 阿饱的猫式调侃——自嘲但不自贬 |

## 核心焦虑

| 焦虑点 | 内容切入点 |
|--------|-----------|
| 工作价值感缺失 | "改了8版PPT最后用回第1版" —— 你的努力去了哪里？ |
| 经济压力（房租/外卖涨价） | "外卖越来越贵，工资纹丝不动" |
| 社交疲惫 | "同事聚餐不想去但不去又奇怪" |
| 自我怀疑 | "30岁了还在做基础岗，我是不是很失败" → "猫35岁还在打猎，你急什么" |
| 健康透支 | "熬夜加班后的第二天是怎么过的" |

## 喜欢的内容类型

| 类型 | 原因 |
|------|------|
| 精准嘴替（高共鸣场景） | 看到=被理解，转发=让朋友也理解我 |
| 治愈向（温暖但不说教） | 需要情绪抚慰，抵触"你应该"的说教 |
| 金句/梗图（可收藏截图） | 可以存起来自己看，不一定要分享，收藏=实用价值 |
| 短小精悍（15-30秒） | 通勤/午休碎片时间刷，太长划走 |
| IP角色日常（连续剧感） | 有陪伴感，像追一个"猫同事"的动态 |

## 不喜欢的内容类型

| 类型 | 原因 |
|------|------|
| 说教/鸡汤 | "你只是不够努力" —— 直接取关 |
| 焦虑营销 | "同龄人已经月入5万" —— 抵触，刷到就划 |
| 过度正能量 | 不真实，和打工人的日常割裂 |
| 信息密度过低的萌宠 | 可爱但无信息增量，刷完无感 |
| 制作过于精良的广告片 | "这是广告吧" —— 警惕心骤起 |

## 内容消费行为

| 平台 | 行为特征 |
|------|----------|
| 小红书 | 搜索行为多（"打工人文案" "治愈插画"），收藏决策强，封面决定点不点 |
| 抖音 | 推荐流刷到，前3秒定生死，完播率和互动率决定是否继续推流 |
| 微信 | 朋友圈转发动因——"想让某些人看到" + "不想直接说的情绪" |

## 粉丝增长路径假设

```
刷到阿饱（封面/前3秒钩子）
  → 有共鸣（"这说的是我"）
    → 点赞/收藏/转发给朋友
      → 朋友刷到 → 循环
    → 点进主页看更多
      → 关注（"想每天看到这只猫"）
```

## 用户分层运营（后续）

| 层次 | 定义 | 运营策略 |
|------|------|----------|
| 粉丝 | 已关注 | 保持日更，评论区互动，维持陪伴感 |
| 路人转粉 | 点赞但不关注 | 在主页设置精选合辑，降低关注决策成本 |
| 传播节点 | 频繁转发分享的人 | 评论区翻牌感谢，鼓励继续转发 |
| 核心粉丝 | 持续评论+私信 | 征求意见、征集选题、共创感 |
