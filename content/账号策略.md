# 猫猫动画IP账号策略

## 一句话定位

**"阿饱是一只又丧又治愈的打工猫，替你说出不敢说的话，再用一碗饭治愈你。"**

---

## 核心人设：阿饱

| 维度 | 设定 |
|------|------|
| 名字 | 阿饱（谐音"吃饱"——再丧也要吃饱，吃饱才能继续打工） |
| 物种 | 橘猫（"大橘为重"天然梗，橘猫=吃+胖=记忆点） |
| 视觉特征 | 胖橘猫、圆脸、豆豆眼、黑眼圈（打工人标配）、小领带（社畜制服） |
| 性格 | 又丧又治愈——吐槽时是打工人嘴替，结尾一定给一勺温暖；不抱怨、不愤怒，用猫的可爱消解丧 |
| 口头禅 | "又是打工的一天喵。" / "累了，但饭还是要吃的。" / "问题不大，先吃饭。" |
| 声音 | 慵懒男声/女声均可，关键是语速慢、带气声，像刚睡醒的猫在说话 |

### 人设禁忌

- 不得使用攻击性语言（不骂老板、不骂同事、不骂社会）
- 不得表达绝望、自暴自弃（丧完必须给温暖出口）
- 不得涉政、涉敏感话题
- 不得过度卖萌（阿饱不是萌宠号，是"披着猫皮的打工人"）

## 内容主线

**一句话主线**：打工猫阿饱的日常——吐槽工作、吃饭治愈、偶尔偷懒。

### 三大内容支柱

| 支柱 | 占比 | 说明 | 示例选题 |
|------|------|------|----------|
| 打工人嘴替 | 45% | 吐槽职场日常，替用户说出心声 | "周一早上闹钟响的那一刻" "PPT改到第8版老板说用回第1版" |
| 吃饭治愈 | 30% | 以食物作为温暖出口，丧完好好吃饭 | "加班到9点，便利店饭团是今晚的MVP" "周末终于给自己做了一顿饭" |
| 猫式哲学 | 25% | 阿饱的"猫生智慧"——用猫的视角重新框定打工人的烦恼 | "猫从不焦虑KPI，猫只焦虑罐头什么时候开" "躺平不是放弃，是保存体力" |

## 适合的内容类型

| 类型 | 适合原因 | 频率建议 |
|------|----------|----------|
| 职场吐槽短动画（15-30秒） | 核心赛道，打工人共鸣强，转发率高 | 每周2-3条 |
| "阿饱吃饭"治愈向（10-15秒） | 视觉记忆点（橘猫+食物），完播率高 | 每周1-2条 |
| 猫式金句卡片（单图+文字） | 制作成本低，适合日更，金句容易截图传播 | 每周3-5条 |
| 阿饱小剧场（30-60秒有剧情） | 深度内容，展现阿饱性格，适合涨粉 | 每周1条 |
| 职场情景配音（阿饱配音+画面） | 蹭热点快，制作快，容易出爆款 | 热点驱动 |

## 不适合的内容类型

| 类型 | 不适合原因 |
|------|-------------|
| 纯萌宠卖萌（无文案无梗） | 和萌宠号正面竞争，丧失"打工人嘴替"差异化 |
| 热点新闻评论（社会/政治） | 风险高，和人设冲突，阿饱不碰严肃话题 |
| 知识科普（职场技巧/效率工具） | 用户期待情绪价值不是知识价值，定位混乱 |
| 剧情反转/悬念类（吊胃口） | 丧+治愈的基调不适配紧张感，破坏"舒服"的品牌感 |
| 带货测评（好物推荐） | 前期不碰变现内容，破坏人设信任 |

## 标题与封面方向

### 标题公式（可套用）

1. **场景代入型**："当____时，你的内心be like："
2. **嘴替型**："打工人周一内心OS（老板勿看）"
3. **金句型**："阿饱说：____"
4. **对比型**："你以为的____ vs 实际上的____"
5. **悬念+治愈型**："加班到崩溃，直到打开这碗面……"
6. **数字清单型**："打工人最崩溃的3个瞬间，谁懂啊"

### 封面原则

- 阿饱脸部占画面50%以上（IP辨识度）
- 表情和标题情绪一致（丧→豆豆眼+叹气 / 治愈→捧着碗闭眼享受）
- 标题大字压在画面上方/下方，字体统一（推荐：粗圆体/旁门正道标题体）
- 配色：暖黄+白+深灰（橘猫天然适配暖色调）
- **禁止**：过度PS、过度精美——保留"手绘感"和粗糙感，拉近距离

## 平台差异策略

| 维度 | 小红书 | 抖音 |
|------|--------|------|
| 内容重心 | 金句卡片 + 治愈图文，强文案导向 | 短动画 + 配音，强节奏导向 |
| 发布频率 | 日更1-2条（金句卡可多更） | 日更1条 |
| 标题风格 | 偏"笔记风"，可用emoji分段，信息密度高 | 偏"对话风"，口语化，5-15字 |
| 封面 | 3:4竖图，标题嵌在封面 | 9:16竖屏，标题在视频前3秒口播 |
| 互动方式 | 评论区引导"你的周一be like？" | 评论区引导"@你的饭搭子" |
| 话题标签 | 3-5个精准标签 | 2-3个热门标签+1个IP标签 |
| 发布时间 | 12:00-13:00 / 18:00-19:00（通勤+午休） | 12:00-13:00 / 18:00-19:00 / 21:00-22:00 |
| 差异化重点 | 封面文字可读性、收藏价值 | 前3秒钩子、完播率 |

## 优先选题方向（P0-P2）

### P0：第一批发必做（3条）

1. **"又是打工的一天喵"** — 阿饱自我介绍+周一综合症（立人设，30秒）
2. **"老板说'辛苦了'，阿饱内心OS"** — 打工人嘴替经典场景（共鸣扩散，20秒）
3. **"加班回家，一个饭团的治愈"** — 丧完吃饭的完整叙事（温暖出口，30秒）

### P1：第2-4周可选

4. **"PPT改到第8版，老板说用回第1版"** — 高共鸣职场吐槽
5. **"猫的KPI只有'今天有没有吃饱'"** — 猫式哲学系列开场

### P2：后续储备

6. "阿饱的年终总结"（年末节点）
7. "打工猫的出租屋改造"（连续剧）
8. "阿饱的同事（其他猫）"（拓展角色宇宙）

## 创作原则

1. **丧不过3秒**：吐槽必须有，但结尾必须有温暖出口——一碗饭、一句话、一个小动作
2. **猫说人话**：站在猫的视角讲人的烦恼，但不能失去猫感（舔爪子、伸懒腰、对罐头执念）
3. **不制造焦虑**：不说"你再不努力就完了"，只说"我懂你，累了就歇会儿"
4. **每期一个钩子**：前3秒必须让用户知道"这说的是我"
5. **可截图传播**：每条出1句金句/1个表情，方便用户截图转朋友圈/发给朋友
6. **视觉一致性**：阿饱形象、配色、字体风格固定，20条内建立辨识度
